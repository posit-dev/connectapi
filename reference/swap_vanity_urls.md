# Swap Vanity URLs

Swap the vanity URLs of two pieces of content.

## Usage

``` r
swap_vanity_urls(content_a, content_b)
```

## Arguments

- content_a:

  A Content object

- content_b:

  A Content object

## Value

A list of the new vanity URLs for `content_a` and `content_b`
