# Get a Variant Schedule

**\[experimental\]** Gets the schedule associated with a Variant.

## Usage

``` r
get_variant_schedule(variant)
```

## Arguments

- variant:

  A Variant object, as returned by
  [`get_variant()`](https://posit-dev.github.io/connectapi/reference/variant.md)
  or
  [`get_variant_default()`](https://posit-dev.github.io/connectapi/reference/variant.md)

## Value

A VariantSchedule object
