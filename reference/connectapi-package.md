# connectapi: Utilities for Interacting with the 'Posit Connect' Server API

Provides a helpful 'R6' class and methods for interacting with the
'Posit Connect' Server API along with some meaningful utility functions
for regular tasks. API documentation varies by 'Posit Connect'
installation and version, but the latest documentation is also hosted
publicly at <https://docs.posit.co/connect/api/>.
